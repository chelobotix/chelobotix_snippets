# Service Vs Lib vs Class

```ruby
#Diferencias principales:

1. `app/services`: Específico para lógica de negocio de la aplicación.
2. `app/classes`: No estándar, usado para clases auxiliares específicas de la aplicación.
3. `lib`: Para código más genérico o reutilizable, no necesariamente específico de la aplicación.
```
