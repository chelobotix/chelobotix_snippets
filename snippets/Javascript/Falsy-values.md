# Falsy values

```javascript
false: The boolean value false.

null: A special keyword representing a null value.

undefined: A special value representing an undefined value.

0: The number zero.

NaN: Not a Number. This is a special value that represents the result of an invalid mathematical operation.

'' (empty string): An empty string is considered falsy.


[] (empty array): true

{} (empty object): true
```
