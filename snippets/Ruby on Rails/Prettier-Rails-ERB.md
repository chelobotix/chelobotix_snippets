# Prettier Rails ERB

```ruby
1. npm install --save-dev prettier @prettier/plugin-ruby prettier-plugin-erb
2. setting.json:
"[erb]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode",
},

3. restart it

4. create .prettierrc in root folder

{
  "printWidth": 80,
  "tabWidth": 2,
  "singleQuote": true,
  "semi": false
}

```
